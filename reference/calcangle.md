# Calculate the Angle Between Two Vectors

This function calculates the angle (in degrees) between two vectors, x1
and x2.

## Usage

``` r
calcangle(x1, x2)
```

## Arguments

- x1:

  A numeric vector.

- x2:

  A numeric vector.

## Value

A numeric value representing the angle between the two vectors in
degrees.

## Examples

``` r
calcangle(c(1, 2), c(2, 2))
#> [1] 18.43495
```
